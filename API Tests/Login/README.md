# Inicia sessió en l'API

Per poder qualsevol operació amb l'API el primer pas sempre
serà iniciar sessió per poder-ne obtenir el token

## Petició

### POST /api-token-auth/

Se li envia l'usuari i la contrasenya a través d'un fitxer
JSON:

```json
{
   "username":"APIm3wi",
   "password":"0BGhwmUWtU9P"
}
```
